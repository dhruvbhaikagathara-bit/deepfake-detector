# Setup Instructions

## For Backend (Member 1 & 2)

1. Install Python (version 3.9 or higher)
2. Go to backend folder: `cd backend`
3. Install requirements: `pip install -r requirements.txt`
4. Create .env file (ask Member 4)
5. Run app: `python app.py`

## For Frontend (Member 3)

1. Install Node.js (version 18 or higher)
2. Go to frontend folder: `cd frontend`
3. Install packages: `npm install`
4. Create .env file (ask Member 4)
5. Run app: `npm start`

## Common Commands

### Git Commands
- Check status: `git status`
- Save changes: `git add . && git commit -m "message"`
- Upload to GitHub: `git push`
- Download latest: `git pull`

### If Something Breaks
1. Check logs in backend/logs/app.log
2. Check browser console (F12)
3. Ask Member 4 (that's me!)
